# AevumDB Documentation

Welcome to the AevumDB documentation!

This section provides comprehensive guides and detailed information about the AevumDB project, including its architecture, API usage, development setup, and more.

## Overview

AevumDB is a high-performance, embedded NoSQL database engine featuring a hybrid kernel. It combines the low-latency I/O and networking of C++17 with the memory safety and complex query logic of Rust.
